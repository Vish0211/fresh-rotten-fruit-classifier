\# 🍎 Fruit Freshness Detection



\## Project Overview



This project is a deep learning-based image classification system that predicts whether a fruit is fresh or rotten. The model is trained using PyTorch and ResNet18 and can classify uploaded fruit images into two categories: Fresh Fruits and Rotten Fruits.



\## Features



\* Upload a fruit image

\* Predict whether the fruit is fresh or rotten

\* Deep learning-based image classification

\* User-friendly Streamlit interface



\## Technologies Used



\* Python

\* PyTorch

\* TorchVision

\* ResNet18

\* Streamlit

\* PIL (Python Imaging Library)



\## How to Run



1\. Clone the repository

2\. Install the required packages

3\. Run the Streamlit application



```bash

streamlit run app.py

```



\## Project Workflow



Dataset Collection

→ Data Preprocessing

→ Train/Test Split

→ ResNet18 Model Training

→ Model Saving (fruit\_classifier.pth)

→ Streamlit Web Application

→ Image Upload

→ Prediction
